
# TaskNest 🪺

**Organize Today, Own Tomorrow.**

TaskNest is a simple command-line to-do list app built in Python. It lets you add, view, complete, and delete tasks — all from your terminal.

## Features

- ✅ Add new tasks
- 📋 View all pending tasks
- ✔️ Mark tasks as completed
- 🗑️ Delete tasks you no longer need
- 🔁 Simple, menu-driven interface

## Getting Started

### Prerequisites

- Python 3.6 or higher

### Run it

```bash
python tasknest.py
```

You'll see a menu like this:

```
===================================
   TaskNest - Organize Today, Own Tomorrow
===================================
1. Add Task
2. Show All Tasks
3. Complete a Task
4. Delete a Task
5. Exit
Enter your choice:
```

Just enter a number (1–5) to navigate the app.

## Example

```
Enter your choice: 1
Enter a new task: Finish homework
Task added successfully!
```
